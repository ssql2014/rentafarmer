# RentAFarmer

RentAFarmer 是一个 Codex skill 和轻量演示项目。用户用中文说明想种的一种或多种蔬菜、地块条件和租赁时长，AI 根据当前季节、面积、光照、土壤、水源和预算生成种植意向建议。用户确认意向后，加微信进入主理人核地块、核农时和最终报价。

## Skill 用法

在 Codex 中使用该 skill 时，让用户提供：

- 想种的蔬菜品种，一种或多种
- 地块面积
- 地区或县域
- 光照、土壤、水源、大棚条件
- 租赁时长或代管周期
- 预算和配送需求

AI 输出推荐品种、面积、租赁时长、费用预估、成熟周期、预计产量、服务配置和微信承接话术。默认服务特色是有机方式管护、不打农药、24 小时视频监控、当地农民提供服务。瓜果成熟后，用户可以自行采摘、邮寄到家，或委托主理人代为销售。

## 运行演示网站

```bash
npm start
```

打开 `http://localhost:4321`。

## 使用 CLI / Script

```bash
node scripts/recommend.mjs "我想种番茄和黄瓜，地块 2 分，全日照，沙壤土，租赁时长 4 个月"
node bin/rentafarmer.js recommend "广东，60 平方米，半日照，想种生菜和菠菜，租期 2 个月"
node bin/rentafarmer.js parse "北方 1 亩地，想种甜玉米和辣椒，有水源，预算 3000，代管半年"
```

## 微信主理人承接

默认微信号是 `rentafarmer-service`。可以用环境变量替换：

```bash
RENTAFARMER_WECHAT=your-wechat-id
```

当前版本不直接引导支付，先让区域主理人加微信对接客户，确认地块、农时、视频监控、配送、成熟后处理方式和合同边界。有机方式管护不等同于有机产品认证；如果用户要求认证级有机产品，需要主理人核验证书。

## API

```http
POST /api/plan
Content-Type: application/json

{
  "text": "我想种番茄和黄瓜，地块 2 分，全日照，沙壤土，租赁时长 4 个月"
}
```

返回推荐品种、面积、租赁时长、费用预估、成熟周期、预计产量、合规边界和微信承接话术。
